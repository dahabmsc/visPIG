visPIG - Visual Plotting Interface for Genetics

1. Upload a regions file

The required format of the file is given below. This will get you started.

2. Upload other data files

Be warned that if you navigate away from the plotting page then the files you've uploaded will require re-uploading the next time you visit the page. There is a 50mb file upload limit. If you have larger files please reduce their size by only including rows that are in the region you are plotting (e.g. by AWK'ing the files).

3. Look at all the paramaters available to edit and change some.

The parameters are categorized by the track they correspond to, or their nature. You can change which parameters you are viewing by using the radio buttons near the top of the page.

4. Save your edited parameters (if you want).

Using the link at the top of the page. Saved parameters can be reapplied by uploading the parameter file from the Files parameter set.
