visPIG - Visual Plotting Interface for Genetics


Note:
The default gene and chromosome sizes files are for the human genome, hg19.
They are provided purely for convenience.
In particular the default gene file should not be regarded as a complete or definite list of the genes in the human genome.
Please download a list of genes appropriate for your data from the UCSC genome browser (or write it yourself).

Release History & News

v1.1 (July 2014)

Various parameters added to the different tracks, as well as an option to show dotted lines across a gap whenever an interaction spans a gap and support added for more general, non-triangular heatmaps.
Also most input files can now be white-space-delimited rather than strictly tab-delimited (three exceptions being the feature track, so that feature annotations can have spaces, the multi-state annotation track(s), which could have spaces on the column with the state ids/names, and the legend file for the multi-state annotation file, which again could have spaces in the state names).

v1.0 (January 2014)

The first public release version of the visPIG web application.
