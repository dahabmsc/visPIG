visPIG - Visual Plotting Interface for Genetics

Creating a plot.

Select the files that you wish to plot on the 'Data Files' tab and then click the big, red 'replot' button to generate a PNG preview of your plot.
Check that your plot looks ok and, if not, adjust the various graphical and track-specific parameters using the other menu tabs, then click 'replot' again.
At the very minimum you will need to provide a regions file, which specifies the region(s) on the genome that you wish to plot.
Please refer to the input data formats page for details on the formats required for the various input files.

Downloading the plot.

Once you have created a perfect-looking plot, either right-click on the PNG preview image and select 'Save Image As' to save your plot as a PNG image, or click the green 'download the PDF' button in the right top panel to generate and download a PDF version of your plot.
The PDF contains a scalable vector graphics version of the plot whereas the PNG is a raster graphics image.
Note that clicking the 'download the PDF' button will mean re-running that the code required to produce your plot is re-run. If your plot is very complicated with lots of data points or lines, you might have to be a bit patient before the download box pops up.

Recreating or modifying a previously created plot.

Once you are happy with your plot, but think you might want to modify or recreate it at a later date, you can save all the parameters that you have set by clicking the yellow 'save the parameters' button in the top right pane.
At a later date you can then recreate the same plot by simply selecting all your upload file and also uploading the saved parameters file (the 'parameter file' in the 'Data files' tab). this will save you the time and trouble to re-set all the parameters you have already set previously.
Note that the parameters get loaded as soon as you select the parameters file. So there is no need to hit 'replot' before changing some of the settings.

Changing the order of tracks.

To change the vertical order of the tracks on your plot you can enter the track name abreviations into the little track order box at the bottom of the 'Data Files' panel.
If you omit a track name completely, it will not be plotted.
You can change the order of tracks for the zoomed part of your plot as well. Simply enter the order of your tracks in the same box under the 'Zoom Parameters' panel.
For track types which can appear multiple times on the plot (e.g. multi-state annotation, intensity, feature or directionality index tracks) you can specify the order of individual instances of that track type by appending the instance number at the end of the track name abbreviation.
Track name abbreviations are given below:
Title Track = title
Heatmap Track = heat
Feature Track = feat (or feat1, feat2, ...)
Directionality Index Track = dix (or dix1, dix2, ...)
Association Track = assoc
Intensity Track = int (or int1, int2, ...)
Interaction Track = hic
Multi-state Track = annot (or annot1, annot2, ...)
phastCons Track = phast
GERP Track = gerp
SNAP Track = snap
Gene Track = gene
Axis Track = axis

Running visPIG locally on your own computer.

Should you not wish to upload your data to the visPIG server, or if some of your files exceed the 50MB upload file size limit, you can download the R code either by clicking the preceding link or by clicking the orange 'download the R code' button in the top right pane on the main visPIG page.
Also download either a default parameter file (simply click the 'save the parameters' button without having changed any) or set some parameters and then download your parameter file.
Instructions on how to run visPIG locally (from the command line) are contained in the R code download.
