visPIG - Visual Plotting Interface for Genetics

The 'Track Code' is the thing you write in the track order box.
e.g. hic,assoc,assoc,axis
It's worth noting that you can repeat tracks. e.g. axis,axis,heat,axis,heat

title

Title track

heat

Heatmap track

feat

Feature track

dix

Directionality index track

assoc

Association track

int

Intensity track(s); NB multiple intensity tracks can optionally be referred to as int1, int2, ...

hic

Interaction track(s); NB multiple intensity tracks can optionally be referred to as hic1, hic2, ...

annot

Multi-state annotation track(s); NB multiple annotation tracks can be referred to as annot1, annot2, ....

phast

phastCons track

gerp

GERP track

snap

SNAP track

gene

Gene track

axis

Axis track

buffer

empty or buffer track (just adds space)
